# Test Card Generator

Website tạo dữ liệu thẻ test với React + Tailwind CSS.

## ⚠️ Lưu ý quan trọng

Đây là công cụ test tạo dữ liệu **KHÔNG HỢP LỆ** với tiền tố `TEST-` và hậu tố `-INVALID`. Dữ liệu này không thể sử dụng trong bất kỳ hệ thống thanh toán thực nào.

## Cài đặt

```bash
npm install
```

## Chạy development server

```bash
npm run dev
```

## Build production

```bash
npm run build
```

## Tính năng

- Nhập BIN (6 số)
- Tạo expiry ngẫu nhiên
- Tạo CVV ngẫu nhiên
- Chọn số lượng thẻ cần tạo
- Hiển thị kết quả dạng bảng
- Copy tất cả kết quả
- Reset form
